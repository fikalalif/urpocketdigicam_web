# UrPocketDigicam Web Profile & E-commerce

Platform profil bisnis dan etalase digital untuk bisnis kamera saku, fokus pada pengalaman visual yang menarik dan interaksi cepat.

## Fitur Utama

* **Dynamic Product Showcase:** Katalog produk dengan filter kategori yang responsif untuk memudahkan navigasi pelanggan.
* **Interactive UI:** Pengalaman pengguna yang mulus tanpa reload halaman yang berlebihan.
* **Brand Identity:** Desain yang disesuaikan dengan estetika brand digital kamera untuk memperkuat citra bisnis.

## Tech Stack

* **Framework:** Laravel
* **Frontend Tech:** Livewire (untuk interaksi reaktif), Tailwind CSS
* **Optimization:** Image Lazy Loading untuk memastikan performa loading yang lebih cepat pada konten visual.

## Highlight Teknis

Saya mengimplementasikan Laravel Livewire untuk membuat komponen pencarian dan filter produk yang reaktif. Pendekatan ini memberikan pengalaman layaknya SPA (Single Page Application) namun dengan struktur backend yang tetap kuat dan reliabel.
